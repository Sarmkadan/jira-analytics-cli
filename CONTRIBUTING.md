# Contributing

We welcome contributions to the Jira Analytics CLI!

To contribute:
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Make your changes and ensure they follow the project coding standards.
4. Add tests for new functionality.
5. Ensure all tests pass (`dotnet test`).
6. Submit a pull request.

Please keep changes focused and documented. By submitting a PR, you agree to license your contributions under the project's MIT license.

For major changes, please open an issue first to discuss the proposed changes.

Thank you for your help!
